The Connect 4 Moderator allows students' artificial intelligence programs to compete against each other and against human players. It's open source, compatible with Mac, Linux, and Windows, and was written by students who previously completed this final project.

# About this Program Test Test This program was created by [Daniel Bulger](http://dpbulger.com) and [Derek Schultz](http://derekschultz.net) for use by students in the Minds & Machines course at Rensselaer Polytechnic Institute. It is a cross-platform alternative to the previous moderator program. You can impress them and earn their respect by finding them and giving them high-fives, or forking this project on Github and contributing your ideas.
# Installing the Connect 4 Moderator Installation packages for ConnectFourModerator are available for Windows, Mac, and Linux. You can also get the source code from Github and compile it yourself if you run [an unsupported operating system](http://plan9.bell-labs.com/plan9/).
